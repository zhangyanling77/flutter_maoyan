# flutter_maoyan
一个flutter app，数据取自猫眼

# 启动项目
```bash

git clone https://github.com/zhangyanling77/flutter_maoyan.git
cd flutter_maoyan
code .
flutter run

```
注意：VS Code要先配置好flutter环境，安装flutter插件，创建好模拟器
